# Templates

Curated app catalog for the platform.

Each template is a directory containing:
- `compose.yml` — Docker Compose spec
- `metadata.json` — name, description, required env vars, exposed port, resource hints
- `icon.svg` — app icon (optional)

Target: ~15 starter templates (WordPress, Nextcloud, Ghost, n8n, Vaultwarden, Plausible, Postgres, Redis, Uptime Kuma, Immich, etc.)
